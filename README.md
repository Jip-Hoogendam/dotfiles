# Dotfiles

My own custom dotfiles i use on a daily basis. It is loosely based on catpucchin
however in a more pastel-pink vibe.

## install guide

First install the required applications. You can change them but you will have
to figure out yourself how.

### Required applications

- [Hyprland](https://hypr.land/)    : Window manager
- [Hyprlock](https://wiki.hypr.land/Hypr-Ecosystem/hyprlock/)    : Lock screen
- [Ghostty](https://ghostty.org/)     : Terminal Emulator
- [zen-browser](https://zen-browser.app/) : Web - browser
- [Dolphin](https://apps.kde.org/dolphin/)     : File Browser
- [Waybar](https://github.com/Alexays/Waybar)      : Taskbar
- [Starship](https://starship.rs/)    : Shell prompt
- [Wlogout](https://github.com/ArtsyMacaw/wlogout)     : Logout screen
- [Fish](https://fishshell.com/)        : Shell
- [Pipewire](https://pipewire.org/)    : audio manger (do not mix with pulse-audio)
- [Slurp](https://gitlab.freedesktop.org/emersion/grim)       : part 1 of screenshot
- [Grim](https://github.com/emersion/slurp)        : part 2 of screenshot
- [Swappy](https://github.com/jtheoof/swappy)      : part 3 of screenshot
- [Sddm](https://github.com/sddm/sddm)        : (optional) display manager
- I forgor        : I'm not perfect. These get it working and in a functional state.

### install

Follow the guide on the arch wiki under the tab for replication
<https://wiki.archlinux.org/title/Dotfiles>;
